> **AiLumi** (meaning "Light") is an empathetic AI companion designed to provide immediate mental wellness support, grounding techniques, and emotional clarity.

## üìñ Overview

In a fast-paced world, access to immediate mental health support is often limited. **AiLumi** bridges this gap by offering a supportive, non-judgmental space for users to express their feelings.

Built with **Google's Gemini API**, Lumi is capable of understanding context, detecting language (English/Malay), and providing actionable advice for anxiety, burnout, sleep issues, and stress management.

**‚ö†Ô∏è Disclaimer:** *AiLumi is an AI support tool, not a replacement for professional medical help. Crisis protocols are hardcoded to direct users to emergency services if severe danger is detected.*

## üöÄ Key Features

### üß† Intelligent Support
* **Contextual Awareness:** Lumi remembers previous parts of the conversation to provide more personalized responses.
* **Multi-Language Support:** Automatically detects and responds in **English** or **Bahasa Malaysia** based on user input.
* **Crisis Intervention Protocol:** Automatically detects keywords related to self-harm or danger and provides emergency contacts (e.g., Befrienders KL).

### üé® User-Centric Design
* **Calm "Lumi Light" UI:** Designed with a soft, warm color palette (Sky Blue & Warm Yellow) to reduce visual stress.
* **Dynamic Quick Actions:** Suggested responses change based on the conversation topic (e.g., if you mention "sleep," the buttons update to sleep-related tips).
* **Mobile Optimized:** Fully responsive design with touch-friendly controls for users on the go.

## üõ†Ô∏è Tech Stack

* **Frontend:** HTML5, CSS3 (Custom "Lumi Light" Theme)
* **Logic:** Vanilla JavaScript (ES6+)
* **AI Model:** Google Gemini 2.0 Flash API
* **Storage:** Local State Management (Session-based context)

## ‚öôÔ∏è Installation & Setup

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/AiLumi-Mental-Wellness-Chatbot.git](https://github.com/YOUR_USERNAME/AiLumi-Mental-Wellness-Chatbot.git)
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd AiLumi-Mental-Wellness-Chatbot
    ```

3.  **Configure API Key:**
    * Open `script.js`.
    * Replace the placeholder `GEMINI_API_KEY` with your own Google Gemini API key.

4.  **Run the application:**
    * Simply open `index.html` in any modern web browser.
